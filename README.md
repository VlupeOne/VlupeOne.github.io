# VlupeOne.github.io

Portfólio pessoal de Thiago Pimentel, publicado como site estático no GitHub Pages.

## Objetivo

Apresentar de forma clara os projetos, a identidade visual e o tipo de trabalho que eu consigo entregar como desenvolvedor em formação.

## O que tem no site

- Seção inicial com apresentação profissional
- Carrossel com projetos em destaque
- Área de contato
- Acessibilidade básica e navegação responsiva
- Layout customizado em HTML, CSS e JavaScript puros

## Projetos em destaque

- [Ticketeria MVP](https://github.com/VlupeOne/ticketeria-mvp)
- [Preço Confirmado MVP](https://github.com/VlupeOne/preco-confirmado-mvp)

## Stack

- HTML5
- CSS3
- JavaScript
- SVG inline
- GitHub Pages

## Publicação

O site é servido diretamente pelo GitHub Pages e pode ser acessado em:

- <https://vlupeone.github.io/>

## Observação

O repositório é intencionalmente simples porque o foco está na apresentação e na clareza do portfólio, não em uma stack complexa.
